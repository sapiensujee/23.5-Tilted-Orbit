# Change Log

All notable changes to the "23.5° Tilted Orbit" theme will be documented in this file.

## [0.1.0] - 2026-06-19

### Added
- **Core Theme**
  - Comprehensive editor UI colors (~200+ color keys)
  - Editor, sidebar, activity bar, tabs, terminal, status bar, breadcrumbs
  - Minimap, peek view, diff editor, merge conflicts, debug toolbar
  - Git decorations, scrollbar, notifications, settings, welcome page

- **Syntax Highlighting**
  - Token colors for all major language scopes
  - JavaScript/TypeScript, HTML/JSX, CSS, JSON, Markdown, Regex
  - Dedicated rules for string interpolation, escape characters, decorators
  - `this`/`self`/`super` keyword styling

- **Semantic Highlighting**
  - LSP-aware token colors for modern editors
  - Bold class/interface declarations
  - Italic parameters, interfaces, and decorators
  - Readonly variables/properties mapped to sunlight gold

- **Terminal**
  - Full 16-color ANSI palette (normal + bright variants)
  - Themed cursor color

- **Bracket Pair Colorization**
  - 6 themed bracket colors from the palette

- **Starfield & Doodles** *(optional, via Custom CSS and JS Loader)*
  - ~35 scattered stars with gold, blue, lavender, pink, and chalk colors
  - Soft glow halos on each star
  - Subtle twinkle animation on select stars
  - 🌞 Sun doodle — center-right, floating
  - 🌍 Earth doodle — center-right, near Sun
  - 🌙 Moon doodle — bottom-right corner
  - 🛸 UFO doodle — bottom-right, near Moon
  - 🌈 Rainbow doodle — top-right area
  - Each doodle has unique floating animation (6–9s cycles)
